## Getting started

The TableSaveViewModal is intended for use together with the Table or StatefulTable component
and the related TableViewDropdown and TableManageViewsModal.

You can add this modal in your project by importing the `TableSaveViewModal`.

```js
import { TableSaveViewModal } from 'carbon-addons-iot-react';
```
