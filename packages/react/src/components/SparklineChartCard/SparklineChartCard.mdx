## Getting Started

The `SparklineChartCard` wraps a Card display around @carbon/charts to render area chart focused on sparkline type.

```jsx
import { SparklineChartCard } from 'carbon-addon-iot-react';
```

## Simple Sparkline Chart

```jsx
import { SparklineChartCard } from 'carbon-addon-iot-react';

<SparklineChartCard
  title="Manage"
  content={{
    xLabel: 'xlabel prop',
    yLabel: 'y label prop',
    xProperty: 'date',
    yProperty: 'value',
    color: {
      pairing: {
        option: 4,
      },
      gradient: {
        enabled: true,
      },
    },
    listContent: [
      { label: 'Target', value: 1000 },
      { label: 'Mean', value: 756 },
      { label: 'Peak', value: 1045 },
    ],
  }}
  values={[
    {
      group: 'Dataset 1',
      date: 1558453860000,
      value: 5,
    },
    {
      group: 'Dataset 1',
      date: 1558453920000,
      value: 5,
    },
    {
      group: 'Dataset 1',
      date: 1558453980000,
      value: 6,
    },
  ]}
  size="MEDIUMTHIN"
  breakpoint="md"
  footerContent={() => <div>Occured on ... </div>}
  isExpanded={false}
  locale="en"
  availableActions={{ expand: true }}
  onCardAction={(id, action) => {}}
/>;
```
